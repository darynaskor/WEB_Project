# WEB_Project: ImageProcessingApp

## Вимоги

### Обмеження
1. Максимальний розмір файлу — **5 МБ**
2. Максимальна кількість операцій в одній задачі — **3**
3. Максимум **3 активні задачі** на одного користувача

### Хід виконання задачі
1. **10%** — завантаження зображення  
2. **30%** — застосування фільтру  
3. **70%** — збереження результату  
4. **100%** — зображення готове  

### Історія задач
Зберігається інформація про:
- назву операції
- статус (`loading`, `in progress`, `done`, `error`)
- результат (посилання на готове зображення)

### Авторизація
- Логін / Реєстрація (**JWT**)
- Кожен користувач бачить **лише свої фото**
